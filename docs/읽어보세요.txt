===========================================
책무기술서 파서 (chaekmu-parser) v0.1.0
===========================================

이 도구는 금융회사 책무기술서 Word 문서(.docx)를
정해진 XLSX 템플릿에 자동으로 옮겨 담는 프로그램입니다.

-------------------------------------------
📦 설치 불필요 — 압축 풀고 바로 실행
-------------------------------------------

1. 받으신 zip 파일을 원하는 폴더에 전체 압축 해제
   예) C:\chaekmu-parser\

   ※ 주의: chaekmu-parser.exe 파일만 따로 옮기면 실행되지 않습니다.
           _internal, templates 폴더가 같은 위치에 함께 있어야 합니다.

2. chaekmu-parser.exe 더블클릭


-------------------------------------------
⚠ "Windows의 PC 보호" 경고가 나오는 경우
-------------------------------------------

처음 실행 시 파란색 경고 창이 뜰 수 있습니다. 이는 이 프로그램이 아직
Microsoft에 공식 등록되지 않은 내부용 도구이기 때문이며, 바이러스가
아닙니다. 다음과 같이 진행하세요.

    [경고 창에서]
        "자세한 정보" 또는 "추가 정보" 클릭
            ↓
        아래쪽에 나타나는 [실행] 버튼 클릭

이 과정은 최초 1회만 필요합니다.


-------------------------------------------
▶ 사용 방법
-------------------------------------------

1. 📄 입력 DOCX — [찾아보기...] 버튼으로 변환할 책무기술서 Word 파일 선택
2. 💾 저장 위치 — 결과 XLSX 파일이 저장될 폴더 지정 (기본: 바탕 화면)
3. [ 변환 실행 ] 버튼 클릭
4. 진행 상황 창에 ✓ 표시가 나오면 완료
5. [ 결과 폴더 열기 ] 버튼으로 저장된 파일 확인

결과 파일명 규칙:  <원본파일명>_output_YYYYMMDD.xlsx


-------------------------------------------
🐞 오류가 발생하면
-------------------------------------------

1. 화면의 [📝 로그 파일 열기] 버튼 클릭
2. 열린 로그 파일(텍스트)을 복사 또는 파일 첨부로 개발자에게 전달
3. 어떤 파일을 변환하려 했는지도 함께 알려주시면 원인 파악에 도움이 됩니다.

로그 파일 위치:
  C:\Users\<사용자명>\AppData\Local\chaekmu-parser\logs\YYYY-MM-DD.log


-------------------------------------------
💡 참고 사항
-------------------------------------------

- 지원 파일 형식(현재): .docx (Word 문서)
- 인터넷 연결 불필요 — 모든 처리는 PC 내부에서만 이루어집니다.
- 입력한 문서는 외부로 전송되지 않습니다.

문의: [개발자 연락처 기재]
