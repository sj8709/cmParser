# 라이나생명 fixture

Phase 2 E2E 회귀 테스트 기준 샘플.

## 필요 파일

| 파일 | 용도 |
|---|---|
| `input.hwp` | 원본 HWP |
| `expected.json` | 기대 출력 (통일 JSON) |

## 구조 변이 메모 (IBK와 다름)

- 임원 17명 / 테이블 75개 / 임원당 테이블 4개
- 회의체: EXEC_INFO 중첩 X, **독립 테이블**로 존재
- 책무: **3열** (IBK는 4열)
- 관리의무: **`<고유 책무>/<공통 책무>` 태그 + ①②③ 원숫자** (IBK는 bold 단락)

상세는 분석문서 섹션 11 참조.
